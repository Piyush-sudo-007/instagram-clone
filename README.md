# Instagram Feed Clone (Flutter)

This project replicates the Instagram Home Feed with pixel-perfect UI and advanced interactions.

## Features

- Instagram style feed
- Stories tray
- Carousel posts
- Pinch to zoom
- Like / Save toggle
- Infinite scrolling
- Shimmer loading state
- Cached network images

## Architecture

Clean architecture with:

models/
repositories/
providers/
services/
widgets/

State management: Riverpod.

Repository pattern used to separate data layer.

## Running the App

flutter pub get
flutter run
